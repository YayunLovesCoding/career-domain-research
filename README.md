# Career Domain Research: Ads vs. RecSys vs. LLMs

Structured research comparing three ML domains from a **5-year career development perspective**.

## Question
As an ML engineer currently in ads, should I stay in ads, move to recommendation systems, or pivot to LLMs?

## Dimensions of Comparison
1. **Market demand** — Hiring trends, team growth, role availability
2. **Compensation trajectory** — Pay trends across domains
3. **Technical innovation velocity** — Research energy, pace of change
4. **Domain convergence** — How ads, recsys, and LLMs are merging
5. **Career optionality** — Transferability of skills, doors opened
6. **Automation / commoditization risk** — Which roles get easier to replace

## Repo Structure
```
sources/corpus.csv          # All curated sources
summaries/ads/              # Per-source summaries (ads domain)
summaries/recsys/           # Per-source summaries (recsys domain)
summaries/llm/              # Per-source summaries (LLM domain)
syntheses/                  # Cross-domain comparative analysis

```
